ВАЖНО: Работаем в папке "Личности" (с русской И)
Полный путь: c:\Users\ilray\Claude\Личности

═══════════════════════════════════════════════════════════
  ШАГ 1: Открой командную строку
═══════════════════════════════════════════════════════════

  Win + R → cmd → Enter


═══════════════════════════════════════════════════════════
  ШАГ 2: Перейди в ПРАВИЛЬНУЮ папку
═══════════════════════════════════════════════════════════

  cd c:\Users\ilray\Claude\Личности


═══════════════════════════════════════════════════════════
  ШАГ 3: Запусти деплой
═══════════════════════════════════════════════════════════

  vercel --prod


  Vercel задаст вопросы:

  ? Set up and deploy?
  → Y (Enter)

  ? Which scope?
  → Выбери свой аккаунт (стрелками, Enter)

  ? Link to existing project?
  → N (новый проект)

  ? What's your project's name?
  → ai-coach-bot (Enter)

  ? In which directory is your code located?
  → ./ (Enter)


  Деплой начался! Жди 30-60 секунд...


═══════════════════════════════════════════════════════════
  ШАГ 4: Получи URL
═══════════════════════════════════════════════════════════

  ✅ Production: https://ai-coach-bot.vercel.app

  Скопируй этот URL!


═══════════════════════════════════════════════════════════
  ШАГ 5: Обновить бота
═══════════════════════════════════════════════════════════

  1. Открой: src/bot/proactive_bot.py

  2. Строка 295, замени:
     webapp_url = "http://localhost:5000"

     на:
     webapp_url = "https://ai-coach-bot.vercel.app"

  3. Сохрани (Ctrl+S)


═══════════════════════════════════════════════════════════
  ШАГ 6: Перезапусти бота
═══════════════════════════════════════════════════════════

  python src/bot/proactive_bot.py


═══════════════════════════════════════════════════════════
  ШАГ 7: Проверь в Telegram
═══════════════════════════════════════════════════════════

  1. Telegram → твой бот
  2. /dashboard
  3. Кнопка "📊 Открыть Dashboard"

  🎉 Готово!
